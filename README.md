## Description

Elise is a tool to generate and import forms to French lexemes on Wikidata.

## Features

### Verbs

* 1st group
  * general case ✓
  * *-cer* ✓
  * *-ger* ✓
  * *-ayer* ✓
  * *-oyer* ✓
  * *-uyer* ✓
  * *envoyer*, *renvoyer* ✓
  * *e* on penultimate syllable ✗
  * *é* on penultimate syllable ✗
* 2nd group
  * general case ✓
  * *fleurir* ✗
  * *haïr* ✗
* 3d group ✗
